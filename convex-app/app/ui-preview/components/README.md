# UI Preview Components

This directory contains AI-generated components for preview and testing.

## How It Works

1. **Generate** - Use slash commands to create components:
   ```
   /ui create a recipe card with image and save button
   /component RecipeCard "displays recipe info"
   /form ContactForm "name email message"
   /page Dashboard "user metrics and activity"
   ```

2. **Preview** - View components at [http://localhost:3000/ui-preview](http://localhost:3000/ui-preview)

3. **Iterate** - Refine with feedback:
   ```
   /ui update RecipeCard: add diet tags and cooking time
   ```

4. **Deploy** - Move to production when ready:
   ```
   Move RecipeCard.tsx to components/ for use in the app
   ```

## Generated Files

Components generated here follow the HealthyMama design system:
- **TypeScript** - Full type safety
- **Tailwind CSS** - Utility-first styling
- **shadcn/ui** - Pre-built UI primitives
- **Convex** - Real-time data integration
- **Clerk** - Auth-aware components
- **Accessible** - ARIA labels and semantic HTML
- **Responsive** - Mobile-first design
